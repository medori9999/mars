# 주식 더미 데이터 시스템 가이드

이 폴더는 시장 탭 및 주식 상세 페이지에서 사용할 수 있는 독립적인 더미 데이터를 포함하고 있습니다.

## 📁 디렉토리 구조 및 위치
- **경로**: `src/data/external_dummy/`
- **구성**:
  - `README_KR.md`: 현재 읽고 계신 설명서 파일입니다.
  - `stock_01.ts` ~ `stock_10.ts`: 각 종목별 개별 데이터 파일입니다. (종목 정보 및 3개년 차트 데이터 포함)
  - `loader.ts`: 개별 파일들을 통합하여 시스템에 제공하는 안전한 로더 파일입니다.

## 🛠 사용 방법
본 데이터는 `src/services/api.ts` 파일에서 백엔드 연결이 원활하지 않거나 테스트가 필요할 때 자동으로 참조되도록 설계되었습니다.

1. **연동**: `loader.ts`에서 각 파일을 동적으로 가져오며, `api.ts`의 `fetchCompanies` 및 `fetchStockChart`에서 이 데이터를 우선적으로 혹은 폴백(fallback)으로 호출할 수 있습니다.
2. **독립성**: 이 폴더 내의 `stock_*.ts` 파일들은 상호 의존성이 없으며, 특정 파일을 삭제하더라도 `loader.ts`가 남은 파일들만 자동으로 인식합니다.

## ⚠️ 삭제 및 초기화 가이드
본 프로젝트의 메인 로직에 최대한 영향을 주지 않도록 설계되었습니다.

- **폴더 전체 삭제 시**: 
  1. `src/data/external_dummy/` 폴더를 삭제합니다.
  2. `src/services/api.ts` 파일 상단에서 `import { dummyLoader } from '../data/external_dummy/loader';` 부분을 주석 처리하거나 삭제합니다.
- **개별 종목 삭제 시**:
  - `stock_01.ts` 등 개별 파일을 삭제하면 `loader.ts`의 동적 탐색(glob) 기능 덕분에 별도의 코드 수정 없이 목록에서 사라집니다.

## 📊 데이터 특징
- **데이터 기간**: 2023년 1월 ~ 2026년 2월 (약 3년)
- **제공 데이터**: 정규장 일시, 시가(Open), 고가(High), 저가(Low), 종가(Close)
- **변동성**: 각 종목은 고유의 변동성을 가지며, 임의의 상한가/하한가 흐름이 포함되어 실감나는 차트 테스트가 가능합니다.
